# PMM Sherpa MCP

PMM Sherpa is a remote MCP server that turns any modern AI client into a senior product marketing partner. It is grounded in 38,000+ curated chunks from books, practitioner AMAs, podcasts, and thought-leader essays, and it speaks the language of positioning, messaging, pricing, GTM, and competitive strategy.

- **Production URL**: `https://pmmsherpa.com/api/mcp`
- **Staging URL**: `https://staging.pmmsherpa.com/api/mcp`
- **Transport**: Streamable HTTP
- **Auth**: OAuth 2.1 with PKCE (sign in with the same Google or email account you use at [pmmsherpa.com](https://pmmsherpa.com))

## The four tools

Once connected, your AI can call these tools:

- **`ask_sherpa`**: chat with the advisor. Best for open-ended PMM questions, framework lookups, and "how would a senior PMM think about this?" prompts.
- **`draft_artifact`**: generate a structured deliverable (positioning statement, messaging house, launch plan, sales narrative, and more) with the framework baked in.
- **`get_feedback`**: paste a draft, a URL, or a competitor's page and get a critique against the same heuristics top GTM leaders use.
- **`scope_pmm_research`**: at the start of a Deep Research run, scope what the planner should focus on so subagents pull the right corpus slices.
